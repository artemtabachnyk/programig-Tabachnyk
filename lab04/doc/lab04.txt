	1. Створення папки з кодом програми 
         
        За допомогою команд копіюэмо потрібні файли до нашого репозиторію 
          sudo cp programig-Tabachnyk/lab03/Makefile programig-Tabachnyk/lab04
          sudo cp programig-Tabachnyk/lab03/README.md programig-Tabachnyk/lab04
        та за допомогою таких команд створюэмо потрыбны нам репозиторії та документи
          mkdir lab04/src
          mkdir lab04/doc
          touch lab04/src/main.c
          touch lab04/doc/lab04.txt
        В нас виходе ось такий репозиторій для лабораторної роботи

        ├── Makefile
        ├── README.md
        ├── doc
        │ └── lab04.txt
        └── src
          └── main.c


	2. Написання коду програми 
           Треба зайти в main.c та написати код програми для задачі "Визначити століття за заданим роком"
           
 #include <stdio.h>
int main()
{
        int xxx = 1986; // Змінна, в яку ми вписуємо придумане число
        int yyy = xxx / 100; //Ввідділяємо ліву частину числа
        int zzz = xxx % 100; // Відділяємо праву частину числа
        int aaa = yyy + 1; //Якщо число перебільшує праву частину то століття буде дорівнювати на 1 більше, томму нам і>        if (zzz == 0) { //Якщо остача від ділення на 100 дорівнює 0
                printf("Століття = %d", yyy); //то виводимо на екран століття
        } else if (zzz > 0) { //Якщо остача від ділення на 100 більше 0
                printf("Століття = %d", aaa); //То виводимо на екран число яке більше від правої частини на 1, тобто ст>        }

        return 0;
}
           

		

	3.Прокомпілювати программу та отримати виконуючий файл

	├── Makefile
        ├── README.md
        ├── dist
        │   └── main.bin
        ├── doc
        │   └── lab04.txt
        └── src
             └── main.c
